Kenapa ping pertama timeout?
Ping pertama biasanya Request Timed Out karena perangkat belum tahu MAC Address tujuan, sehingga harus melakukan proses ARP (Address Resolution Protocol) terlebih dahulu sebelum mengirimkan ICMP Reply.
